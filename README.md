# Inventory-management-system

Requirements :

1.Flask
 
  Install Flask & Flask-SQLAlchemy :
  
     1. pip install Flask
     2.pip install Flask-SQLAlchemy
 
2. Python Idle 3.6
3. Connection db 

4. Create HTML files  :
      
     1.base
     2.index(main file)
     3.location - Update 
     4.product - Update
     5.movements - Update
     6.Report

5. Run the code :

   1. Open cmd inside the folder 
   2. to run the application :
         flask run
   3. http://127.0.0.1:5000/ open this server in the browser( chrome )
